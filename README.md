# Weather Diary

**Автор:** Трушин Дмитрий
## Описание

Программа для ведения дневника погоды. Позволяет добавлять записи с датой, температурой, описанием погоды и осадками. Можно фильтровать записи по дате и температуре. Все данные сохраняются в JSON-файл.

## Как запустить

1. Склонируйте проект или скачайте исходный код.
2. Убедитесь, что Python установлен.
3. Запустите файл `main.py` командой:  
   `python main.py`

## Использование

- Введите дату, температуру, описание погоды, отметьте осадки при необходимости.
- Нажмите "Добавить запись".
- Используйте фильтры для поиска по дате или температуре.
- Все записи сохраняются автоматически при закрытии программы.

## Примеры тестов

- Добавить запись: дата - 2024-05-20, температура - 15, описание - "Облачно", осадки - да.
- Фильтр по дате: 2024-05-20.
- Фильтр по температуре: больше 10°C.

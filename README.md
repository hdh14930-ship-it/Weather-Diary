Weather Diary (Дневник погоды)

Разработать GUI-приложение "Weather Diary", позволяющее добавлять, просматривать и фильтровать записи о погоде, сохранять данные в JSON и использовать Git.

Пошаговая инструкция:

1. Создайте интерфейс с полями для ввода:

○ Дата

○ Температура

○ Описание погоды

○ Осадки (да/нет)

2. Реализуйте кнопку "Добавить запись", которая добавляет новую запись в список или таблицу.

3. Добавьте возможность фильтрации:

○ По дате (например, выбрать записи за определённый день)

○ По температуре (например, показать записи выше +10°C)

4. Реализуйте функцию сохранения записей в файл JSON и загрузки из файла.

5. Проверяйте корректность ввода:

○ Дата должна быть в правильном формате.

○ Температура — число.

○ Описание не должно быть пустым.

6. Используйте Git:

○ Создайте локальный репозиторий.

○ Добавьте .gitignore.

○ Выложите проект на GitHub.

7. Напишите README (в формате .txt, .md или .docx), в котором укажите:

○ Имя и фамилию автора

○ Краткое описание программы

○ Как запустить приложение

○ Примеры использования (возможные тесты — по желанию)
